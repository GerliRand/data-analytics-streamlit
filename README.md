# Data Analytics - Streamlit app
This project is part of a Data Analytics course assignment.

## Description
The app vizualize hotel statistics in Finland using Statistics Finland municipality-level hotel stats from: 
https://pxdata.stat.fi/PxWeb/pxweb/en/StatFin/StatFin__matk/statfin_matk_pxt_11lm.px/

## Features 
- Data loading (CSV)
- Data cleaning and preprocessing
- Time series visualization (line charts)
- Yearly aggregation (bar chart)
- Comparison between municipalities (Espoo vs Vantaa)
- Download button for exporting data (CSV)

## Technologies
- Python (3.12.10)
- Pandas
- Streamlit

## How to run
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
